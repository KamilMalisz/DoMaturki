# Utworzenie nowego projektu vite od zera:
1. Nowy projekt vite w aktualnym folderze:   npm create vite@latest ./
2. Wybranie opcji React i JavaScript
3. npm install
4. npm run dev


# W Visual Studio Code instalujemy rozszerzenie Prettier
# Po instalacji przechodzimy do menu file->Prefferences->Settings
# i wyszukujemy "format", upewniamy się domyślnie zaznaczony 
# jest Prettier w opcji Editor Default Formatter.
# Następnie szukamy "indent" i zaznaczamy 2 
# w opcji Editor Tab Size, dzięki temu tabulacja
# będzie miała dwa znaki we wszystkich plikach
